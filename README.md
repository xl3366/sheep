## 羊了个羊 sheep

羊了个羊 无需抓包一键通关

链接：https://sheep.idcw.eu.org

## 教程

1.打开微信羊了个羊小游戏

<img src="https://dd-static.jd.com/ddimg/jfs/t1/114123/26/29811/187960/6326d7cfE5d00dd97/ccbef7e8c6b63ea2.png" alt="img" style="zoom: 25%;" />

2.复制头像下面的uid

3.输入到网页端的UID 输入框

4.点击获取token（如果获取不到，可以参考下面抓包教程）

5.设置入群次数

6选择游戏模式

7.输入每局次数（选填，为空则随机）

8.点击一键入群按钮
